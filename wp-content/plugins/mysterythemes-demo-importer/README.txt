One Click Demo Importer For Mystery Themes official themes demo content, customization options, widgets and theme settings.
This plugin fetches the mysterythemes github config files to get demo source details. 

== Description ==

Import "Mystery Themes" every official themes by only a single click. Simple installation and activation of all required plugins at occurrence. Import all the customization options, theme setings, widgets of official themes in a simplicity way.

= Requirements =

* WordPress 4.7 or later.
* [Mystery Themes Official Themes](https://mysterythemes.com/themes/).

== Installation ==
    1. Login to your WordPress Admin Area
    2. Go to Plugins > Add New
    3. Type “Mystery Themes Demo Importer” into the Search and hit Enter.
    4. Find this plugin Click “Install now”
    5. Activate The Plugin
    6. Use Mystery Themes Demo Importer on any themes from mysterythemes to import demo.

== Manual Installation ==
1. Download the plugin from WordPress.org repository
2. On your WordPress admin dashboard, go to ‘Plugins > Add New > Upload Plugin’
3. Upload the downloaded plugin file (mysterythemes-demo-importer.zip) and click ‘Install Now’
4. Activate ‘Mystery Themes Demo Importer’ from your Plugins page.
5. Use Mystery Themes Demo Importer on any themes from mysterythemes to import demo.
